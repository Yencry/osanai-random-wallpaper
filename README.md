# Osanai Random Wallpaper

一个基于小佐内由纪的纯前端的随机壁纸小站，每次打开页面都会展示一张随机壁纸，并支持完整轮询和多图片格式优化（AVIF / WebP / JPG / PNG）。

## 功能

- **随机壁纸首页（index.html）**
  - 采用完整轮询 + 洗牌算法：在播完一整轮之前不会重复同一张图片。
